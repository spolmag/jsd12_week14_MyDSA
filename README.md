# MyDSA - JavaScript Array Methods Practice

A practice project focused on mastering essential JavaScript array methods: `map()`, `filter()`, and `find()`. The exercises use data from "Jojo's Bizarre Adventure" to make learning more engaging.

## Purpose

The goal of this project is to understand how to efficiently manipulate and query arrays in JavaScript using functional programming patterns.

## Project Structure

- `jojo.js`: Contains the sample dataset of characters (`jojoCharacters`).
- `exercise_1.js`: Implementation file for practice exercises.
- `exercise_2.js`: Additional practice exercises.
- `my-understanding.md`: Personal notes on sorting algorithms, binary search, and array methods.

## Methods Covered

- **`map()`**: Creates a new array by calling a function for every array element.
- **`filter()`**: Creates a new array with all elements that pass the test implemented by the provided function.
- **`find()`**: Returns the value of the first element in the array that satisfies the provided testing function.

## How to Run

To see the results of the exercises, run the files using Node.js:

```bash
node exercise_1.js
node exercise_2.js
```
